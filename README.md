# 🩺 Diabetes Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict whether a patient has diabetes based on medical features using machine learning classification models.

---

## 🎯 Objectives

* Clean and preprocess medical data
* Train classification models
* Compare model performance
* Evaluate predictions using standard metrics

---

## 📊 Dataset

The dataset contains medical features such as:

* Glucose
* Blood Pressure
* BMI
* Insulin
* Age

Target:

* Outcome (0 = No diabetes, 1 = Diabetes)

---

## 🔍 Data Preprocessing

* Replaced zero values with NaN
* Handled missing values using mean imputation
* Applied MinMax scaling

---

## 🤖 Models Used

* Logistic Regression
* KNN
* SVM
* Random Forest
* Decision Tree

---

## 📈 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score

---

## 🏆 Results

Logistic Regression achieved the best overall performance among tested models.
